This will be pretty much exactly the same, differing only in the details (what options appear on the side, columns in the tracker, CSVs to be downloaded)

  • Data Download: Pretty much as shown
  • Options:

    • Blocking: Same structure as in the screenshot, but with the blocks of this study (silent, practice, etc.). The IAT will be separated from the test blocks.
